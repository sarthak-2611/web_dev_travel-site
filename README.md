# Travel Landing Page

This is a simple responsive travel website landing page made using only HTML and CSS.  
It was created as a college assignment to practice basic web development and layouts.

## Features

- Navbar with links to different sections  
- Hero section with main heading and image  
- Services / categories section  
- Top destinations cards  
- Steps to book a trip  
- Testimonials section  
- Partner brands section  
- Newsletter subscription form  
- Footer with basic info and links  
- Responsive design for mobile and tablet using media queries

## Technologies Used

- HTML5  
- CSS3 (Flexbox and Grid)
